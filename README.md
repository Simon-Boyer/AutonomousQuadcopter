Quadcopter autonome
2015
-------------------------------

Projet qui permet de un déplacement autonome du drone, en suivant les tracés des rues
Permet de se déplacer en ville plus facilement

Nécessaire:
-Arduino Mega
-Aeroquad kit et modules
-GPS module
-Drone sans contrôleur

Configuration:
1.Installer tous les modules sur aéroquad
2.Installer module GPS sur aéroquad
3.Installer aéroquad sur arduino
4.Installer le code dans dossier Aeroquad_v3.2 sur l'arduino
5.Installer arduino sur le quad
6.Démarrer une serveur npm avec "Directions"
